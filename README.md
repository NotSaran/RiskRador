
# RiskRador

## AI-Powered Loan Default Prediction System

RiskRador is a Machine Learning project that predicts whether a customer is likely to default on a loan using the Home Credit Default Risk dataset.

---

## Project Objective

Develop a predictive model that helps financial institutions assess customer credit risk and reduce loan defaults.

---

## Dataset

Home Credit Default Risk Dataset

Rows: 307,511

Features: 109
